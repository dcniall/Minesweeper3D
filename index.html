Thumbtack Minesweeper3D

Size (Cubed) Bombs

Cheats:

* To Rotate - Click and drag anywhere
* Right-Click on a block to set a flag (Thumbtack) ;)
* Left-Click on a block if your feeling lucky!
* Scroll-Wheel zooms
* Click the grid to start

This example requires a browser that supports the HTML5<canvas> feature.
